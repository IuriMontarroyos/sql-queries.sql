# Análise de Vendas – Power BI e SQL

## Objetivo
Projeto de análise de dados com foco em apoio à tomada de decisão, utilizando dados simulados
de um e-commerce pet. O objetivo é analisar receita, ticket médio e comportamento de clientes
por categoria e localização.

## Estrutura dos Dados
Os dados foram organizados em três tabelas:
- clientes: informações de localização e tipo de cliente
- produtos: catálogo de produtos por categoria
- vendas: registros de pedidos, quantidade e receita

## Tecnologias Utilizadas
- Power BI (dashboards e visualizações)
- SQL (consultas analíticas)
- CSV (fonte de dados)
- Git/GitHub (versionamento)

## Análises Realizadas
- Receita total
- Ticket médio por pedido
- Receita por categoria de produto
- Receita por estado
- Evolução da receita ao longo do tempo

## Observações
Os dados utilizados são fictícios e foram criados para fins de estudo e portfólio.
